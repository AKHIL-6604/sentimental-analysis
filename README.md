# sentimental-analysis




This project presents an advanced sentiment analysis system that understands both textual content and emojis using deep learning techniques. Traditional sentiment analysis models mainly focus on text and often ignore emojis, which carry strong emotional meaning in social media communication.
The proposed system integrates text and emoji information to improve sentiment prediction accuracy. Using Transformer-based models like BERTweet, the system classifies sentiments into:
Positive 😊
Negative 😡
Neutral 😐
The project is designed for analyzing social media content, customer reviews, and online feedback where emojis play a significant role in expressing emotions.
